## 📋 本次更新内容

### 1. 功能改进

#### ✅ 学生总分导出格式修复
- **问题：** 导出 Excel 时得分显示为小数（如 100.00）
- **解决：** 改为整数显示（如 100）
- **文件：** `routes/admin.js` 第 708 行

#### ✅ 班主任删除时保留积分记录
- **问题：** 删除班主任后，其积分记录被级联删除
- **解决：** 修改外键策略为 `ON DELETE SET NULL`
- **迁移脚本：** `migrate_score_records_keep_on_teacher_delete.js`
- **执行方式：** `node migrate_score_records_keep_on_teacher_delete.js`
- **已执行状态：** ✅ 已在数据库中应用

#### ✅ 批量导入班级号码补零
- **问题：** 输入「01」被转换为「1」
- **解决：** 改用文本输入，计算最长号码长度进行补零
- **支持场景：**
  - 前缀「2026」+「01」～「05」 → 202601班～202605班
  - 无前缀+「663」～「730」 → 663班～730班

#### ✅ 小程序登录错误提示
- **问题：** 登录失败显示「HTTP Error: 401」
- **解决：** 改为明确提示「账号或密码错误」
- **文件：** `minipro_ScoreManagement/utils/api.js`

#### ✅ 导入学生错误提示详情
- **问题：** 导入失败时仅显示简要摘要
- **解决：** 改为模态框展示完整错误列表（逐行）
- **文件：** `public/admin/js/app.js`

#### ✅ 学期默认值
- **实现：** 小程序 picker 默认选中已激活学期
- **方式：** 后端 GET `/api/teacher/semesters` 返回按 `is_active DESC` 排序
- **文件：** `routes/teacher.js`

#### ✅ 班主任密码明文显示
- **实现：** 配置班主任时密码显示为明文（type="text"）
- **文件：** `public/admin/js/app.js`

#### ✅ 导出类型选择
- **实现：** 导出 Excel 可选「积分记录」或「学生总分」
- **文件：** `public/admin/js/app.js` 和 `public/admin/js/api.js`

## 🚀 启动与部署

### 快速启动（3 条命令）

```bash
# 1. 初始化数据库（首次）
mysql -u root -p < database.sql

# 2. 启动后端
npm start

# 3. 打开管理后台
浏览器访问 http://localhost:3000/admin
```

### 迁移升级（现有系统）

若系统已运行但需应用删除班主任保留积分的策略：

```bash
node migrate_score_records_keep_on_teacher_delete.js
```

使用微信开发者工具打开 `D:\minipro_ScoreManagement`
