## ⚠️ 重要提醒

### 安全事项
1. **修改默认密码** - 生产环境前必须修改 admin 账户密码
2. **修改 JWT_SECRET** - `.env` 中的 JWT 密钥需要改为强密钥
3. **启用 HTTPS** - 生产环境必须使用 HTTPS

### 开发注意
1. **保持服务运行** - 确保 `npm run dev` 命令一直运行
2. **Token 管理** - 在所有请求中包含 `Authorization: Bearer <token>`
3. **权限隔离** - 班主任只能看自己班级的数据
