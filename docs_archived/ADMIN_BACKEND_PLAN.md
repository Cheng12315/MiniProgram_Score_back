## 🔐 权限控制

### 访问权限

| 功能 | 权限 | 说明 |
|------|------|------|
| 登录 | 所有人 | 无限制 |
| 仪表板 | 登录用户 | 需要 Token |
| 班主任管理 | 仅 admin | 只有管理员可访问 |
| 学生管理 | 仅 admin | 只有管理员可访问 |
| 班级管理 | 仅 admin | 只有管理员可访问 |
| 学期管理 | 仅 admin | 只有管理员可访问 |
| 数据导出 | 仅 admin | 只有管理员可访问 |
| 操作日志 | 仅 admin | 只有管理员可访问 |

### 实现方式

```javascript
// 路由保护
const router = createRouter({
  routes: [
    {
      path: '/admin/login',
      component: Login,
      meta: { requiresAuth: false }
    },
    {
      path: '/admin/dashboard',
      component: Dashboard,
      meta: { requiresAuth: true, role: 'admin' }
    },
    // 其他路由...
  ]
});

// 路由守卫
router.beforeEach((to, from, next) => {
  const token = localStorage.getItem('token');
  if (to.meta.requiresAuth && !token) {
    next('/admin/login');
  } else {
    next();
  }
});
```

## 📁 后端 API 支持情况

### 管理员相关 API（已实现）

```
POST   /api/auth/login                    # 登录（支持管理员）
GET    /api/admin/classes                 # 获取所有班级
GET    /api/admin/semesters               # 获取所有学期
GET    /api/admin/students                # 获取所有学生
POST   /api/admin/students/import         # 导入学生
GET    /api/admin/export/students         # 导出学生数据
GET    /api/admin/export/score-records    # 导出积分记录
```

### 可能需要添加的 API（扩展）

```

GET    /api/admin/teachers                # 获取所有班主任
POST   /api/admin/teachers                # 添加班主任
PUT    /api/admin/teachers/:id            # 修改班主任
DELETE /api/admin/teachers/:id            # 删除班主任

# 班级管理
POST   /api/admin/classes                 # 添加班级
PUT    /api/admin/classes/:id             # 修改班级

# 学生管理
POST   /api/admin/students                # 添加学生
PUT    /api/admin/students/:id            # 修改学生
DELETE /api/admin/students/:id            # 删除学生
```

   - 🔴 高：登录、班主任管理、学生导入、学期管理
