## 测试 3️⃣：班主任导入学生数据

**权限**：班主任只能导入自己班级的学生

### 第一步：准备 Excel 文件

创建一个新的 Excel 文件（或 CSV），内容如下：

```
姓名,学号,班级,性别
白居易,2022006,8年级1班,M
李贺,2022007,8年级1班,F
```

**重要**：
- 必须有列名：姓名、学号、班级、性别
- 班级必须与班主任的班级一致（8年级1班）
- 性别：M（男）或 F（女）

### 第二步：在 Apifox 中上传

```
请求方法: POST
URL: http://localhost:3000/api/teacher/students/import
Header: Authorization: Bearer <班主任token>

Body:
1. 选择 form-data
2. 添加字段：
   - Key: file
   - Value: [选择你的 Excel 文件] （选择 File 类型）
3. 点击 Send
```

### 预期响应
```json
{
  "code": 0,
  "message": "导入完成",
  "data": {
    "successCount": {
      "added": 2,
      "updated": 0
    },
    "errors": [],
    "hasErrors": false
  }
}
```

**检查清单**：
- [ ] 请求成功（Status 200）
- [ ] `code` 为 0（成功）
- [ ] `successCount.added` 为 2（添加了 2 个新学生）
- [ ] `errors` 为空数组

### 验证导入结果

再次查询班主任的学生列表：

```
请求方法: GET
URL: http://localhost:3000/api/teacher/students?page=1&pageSize=10
Header: Authorization: Bearer <班主任token>
```

应该看到：
- 之前的 5 个学生
- **新增**：白居易 (2022006)
- **新增**：李贺 (2022007)
- 总共 7 个学生

## 💡 Apifox 文件上传的正确方式

### 视觉步骤指南

```
┌─────────────────────────────────────┐
│ Apifox 请求界面                      │
├─────────────────────────────────────┤
│ 方法: POST                          │
│ URL: http://localhost:3000/api/...  │
├─────────────────────────────────────┤
│ [Params] [Body] [Header] ...        │
│           ↓点击                      │
│        ┌─────────────────────────┐  │
│        │ Body 选项              │  │
│        │ ☐ raw                 │  │
│        │ ☑ form-data (选这个!)│  │  ← 关键！
│        │ ☐ x-www-form...       │  │
│        └─────────────────────────┘  │
│        ↓                             │
│        ┌─────────────────────────┐  │
│        │ Key | Type  | Value    │  │
│        │ file| File  | [选择]   │  │  ← Type 选 File
│        │     |       | 1.xlsx   │  │
│        └─────────────────────────┘  │
│                                     │
│ [Send 按钮]                         │
└─────────────────────────────────────┘
```

### 常见错误

❌ **错误 1**：使用 raw JSON
```json
{
  "file": "test.xlsx"
}
```

❌ **错误 2**：使用 URL-encoded
```
file=test.xlsx
```

❌ **错误 3**：Type 选错了
```
Key: file
Type: Text  ← 错的！应该选 File
Value: test.xlsx
```

✅ **正确方式**：
```
Key: file
Type: File  ← 关键！
Value: [实际选择文件]
```

## ❓ 关于导出文件返回 XML

### Q: 为什么看到 XML 而不是直接下载文件？

A: 这是正常的！

- 在 Apifox 中，会显示响应的原始内容
- Excel 文件实际上是 ZIP 压缩包，里面包含 XML
- 在真实的浏览器或客户端中，会自动触发下载
- **数据是正确的**，证明后端代码工作正常

### Q: 如何验证导出内容是否正确？

A: 查看 XML 中的 `<v>` 标签：

例如在响应中搜索：
```xml
<v>张三</v>      ← 学生名字
<v>102.00</v>    ← 分数（102 是正确的：100+2）
```

如果这些数据都在，说明导出功能完全正常。
