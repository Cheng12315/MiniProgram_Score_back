## 🔧 常见任务一览

### 系统安装与启动

```bash
# 1. 初始化数据库
mysql -u root -p < database.sql

# 2. 启动后端服务
npm start

# 3. 打开管理后台
浏览器访问 http://localhost:3000/admin
```

### 数据迁移（现有系统）

```bash
# 升级外键策略，保留班主任积分记录
node migrate_score_records_keep_on_teacher_delete.js
```

### 数据备份与恢复

```bash
# 备份
mysqldump -u root -p score_management > backup.sql

# 恢复
mysql -u root -p score_management < backup.sql
```

## ❓ 常见问题快速解答

### Q: 系统无法启动？
**A:** 查阅 [QUICK_REFERENCE.md - 故障排查](./QUICK_REFERENCE.md#🔍-故障排查)

### Q: 如何导入学生？
**A:** 查阅 [DEPLOYMENT_AND_USAGE_GUIDE.md - 学生管理](./DEPLOYMENT_AND_USAGE_GUIDE.md#3-学生管理)

### Q: 删除班主任后积分记录消失了怎么办？
**A:** 
1. 停止服务
2. 执行迁移脚本：`node migrate_score_records_keep_on_teacher_delete.js`
3. 重启服务
4. 后续删除班主任时，积分记录会保留

### Q: 如何导出数据？
